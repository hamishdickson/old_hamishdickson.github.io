What interests me

I've recently finished Andrew Ng's excellent Machine Learning coursera course. I found this fascinating and would love to know more or work on an interesting machine learning problem (I've been trying to find something open source to contribute to, but so far haven't found anything that catchs my eye).

I'd really like to know more about genetics, how the brain works and biology in general - I think this area has a lot of untapped potential for interesting maths and computer modelling.

I did physics at university and still absolutely love it. I still read a lot of physics and maths.
